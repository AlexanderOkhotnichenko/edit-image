# Редактор изображений - Edit Image

Приложение для редактирования изображений, с возможностью сохранить измененное изображение у себя на локальном компьютере.

Формат файлов: 
* svg
* png
* jpeg
* jpg
* webp
* ico

Stack:
* JavaScript
* SCSS
* HTML
* WEBPACK
